# Menu Portal Guide

**Trigger:** Need to create or edit menu items, options, or categories in the Vendor Portal.

**Check First:**
- Access Vendor Portal at https://vendors.valleyeats.ca/ with dispatch credentials
- Restaurant and market selection

**Steps:**
1. **Navigation:** select market > restaurant > click "Menu" on left side. Items organized by category tabs. Use Ctrl+F to search.
2. **Add category:** click "Add Tab" under Menu header. Avoid special characters.
3. **Create item:** determine if it contains alcohol (critical). Add New Item > name, description, price, cuisine type, item type (e.g., "beverages"), menu modifier icons (gluten-free, availability, etc.) > Save.
4. **Create option (modifier):** click Options icon on item > Create New Option > name + price > press "+" > configure Exclusive (one choice only) and Required (must select) settings > Save.
5. **Option sorting:** alphabetical by default. Use numbering (1), 2), 3)) or lettering (a), b), c)) prefixes to control order.
6. **Critical rules:** NEVER delete items (always pause/disable). NEVER delete item photos. Always Save before leaving. Always set items as taxable in Food Details. Always mark alcohol correctly.
7. **Options cannot be disabled, only deleted** -- always note in ticket to restaurant relations when deleting options.

**Escalate If:**
- Need to add new items or restructure menu (Growth team work -- growth@valleyeats.ca)
- Alcohol flag uncertainty

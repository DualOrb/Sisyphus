# Menu Portal Guide

How to use the Menu Portal to create items, options, and manage restaurant menus.

## Accessing the Menu Portal

The Menu Portal is accessed through the Vendor Portal at https://vendors.valleyeats.ca/. Log in with your dispatch credentials.

1. Select the market, then select the restaurant.
2. On the restaurant's Profile & Settings page, click "Menu" on the left-hand side.

Within the menu, items are organized into categories (tabs) on the left. Use Ctrl+F to quickly search through menus.

## Adding a New Menu Tab (Category)

Under the Menu header, locate the "Add Tab" icon and create a new title for the section. Avoid special characters. Example: "Appetizers."

Once the tab is created, you can start adding menu items that fall under that heading.

## Create New Item

1. Determine if the item contains alcohol. This is critical. If an alcoholic beverage is not marked as such, we might not be able to deliver it.
2. Click "Add New Item."
3. Enter a name for the item.
4. Add a description (how the item is prepared, number of items, etc.).
5. Add a price for the item.
6. Set the cuisine type.
7. Adjust the Item Type to match the item. Example: if it is a drink, the item type would be "beverages."
8. Apply any relevant menu modifier icons. Example: "Modifier - is this item Gluten-Free?" or "Availability - is this menu available all day every day or just Tuesdays?"
9. Click Save.
10. Create more menu items if needed.

To add another new item, click "+ Create New Item" below the tab.

## Create New Option

Options are modifiers added to menu items. Use this for side choices, adding or removing toppings, extras, etc.

1. Click the "Options" icon on the menu item.
2. Click "Create New Option."
3. Name the option (e.g., "Extras" or "Sides").
4. Add the modifier name and its price.
5. Press the "plus" icon to add it to the item's option list.
6. Configure the option settings:
   - **Exclusive**: Enabling this means the customer can only choose ONE of the options in the list. Leaving it off means they can choose multiple.
   - **Required**: Enabling this means the customer MUST make a selection from the list before they can place the order.
7. Click Save to save the new modifier.
8. Add more options if needed.

### Option Sorting

Menu options are sorted alphabetically. To control the display order, use a numbering scheme:

For 1 to 9 options, prefix with numbers:
- 1) Option 1
- 2) Option 2
- 3) Option 3

For more than 9 options, prefix with letters:
- a) Option 1
- b) Option 2
- c) Option 3

## Menu Item Functions and Widgets

Each menu item has a row of function icons. Here is what each one does:

### Options Icon
Configures the item's additional options (size, add-ons, flavors, etc.). Clicking it opens the current options for the item. You cannot disable these options -- they can only be deleted. Always note in a ticket to restaurant relations when deleting options.

### Quantity Icon
Sets the available quantity of an item. The count decrements when someone orders it. Most restaurants set this number very high since they are unlikely to run out.

### Prep Time Icon
Sets the additional preparation time needed for this specific item. If the prep time is greater than the restaurant's default delay, the system will push the order back so the restaurant has sufficient time to prepare.

### Copy Icon
Copies the current menu item. There is a separate paste button within the menu to paste it. This is useful for building out menus efficiently.

### Delete Icon
Deletes the item permanently. DO NOT USE. Never delete items. Always disable/pause instead. It is much more work to re-create an item than to re-enable a paused one.

### Availability Icon
Sets which days and times this item is available. Used when a restaurant only prepares certain items on certain days. Also used for alcoholic items that may have time-restricted availability.

### Picture Icon
Deletes the menu item photo. DO NOT USE.

### Food Details Icon
Specifies item attributes: alcohol, gluten, halal, etc. All items created should be set to "taxable" so we are not losing money.

### Copy URL Icon
Copies the URL of the menu item to your clipboard. Not used by dispatchers.

### Save Icon
Saves any changes made to the item back to our systems. If you leave without saving, all changes will be lost. ALWAYS SAVE BEFORE LEAVING.

## Important Rules

- **Never delete items.** Always disable/pause them instead. Even if a restaurant requests deletion, only disable it and create a ticket for growth@valleyeats.ca to confirm the removal.
- **Always save.** Changes are lost if you navigate away without saving.
- **Always set items as taxable** in Food Details so revenue is not lost.
- **Always mark alcohol correctly.** Failure to do so may prevent delivery.
- If a customer already has a disabled item in their cart or has already placed an order with it, the order will have to be manually fixed.

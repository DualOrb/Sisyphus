# Tablet Operations Guide

**Trigger:** Reference for day-to-day tablet functions used by restaurant partners.

**Check First:**
- Which tablet operation is needed (halt, ETA, messenger, orders)

**Steps:**
1. **Halt:** stops new orders. Resets automatically next morning. Restaurant presses Halt icon on tablet, or dispatch changes status to "Halted" on restaurant page. Inform restaurant they can self-halt.
2. **ETA:** adjusts estimated prep time, affecting courier arrival and customer time estimate. Only adjust when necessary.
3. **Messenger:** communication between dispatch and restaurant. Use for: driver delayed, order needs delay/speedup, order needs confirmation. Always call for urgent issues -- restaurants may not check tablet.
4. **Profile:** restaurant can change hours and access web portal for menu changes. Always hit Publish to save.
5. **Orders/History:** shows last 10 days of orders for reference.
6. **Receiving orders:** confirm order, verify ready time (preorders in blue), verify quantities, call support for substitutions/time adjustments, adjust ETA for multiple orders. Package with Order ID label, attach receipt. Mark "ready" for pickup orders.
7. **Restaurant support:** 1-855-610-3287 for substitutions, out-of-stock, driver issues, tablet/internet issues, general support.
8. **Orders not showing:** refresh icon (top right) > power cycle tablet. If still not working, follow tablet-troubleshooting.md.

**Escalate If:**
- Tablet unresponsive after basic troubleshooting
- Restaurant has persistent operational issues

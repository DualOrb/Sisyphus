# Courier Best Practices

**Trigger:** Reference when evaluating driver behavior or giving feedback.

**Check First:**
- What specific behavior needs evaluation or correction?

**Steps:**
1. **Status marking rules:** En-Route = only when actually heading to restaurant (triggers restaurant notification). At Restaurant = only when physically inside waiting. In Bag = once food is in possession; leave as In Bag if multiple orders, only one In Transit at a time. In Transit = only when physically leaving with that order. Delivered = once handed off.
2. **Pickup/delivery protocol:** arrive a few minutes before pickup time. Always use delivery bag. Confirm order number and contents at restaurant. Double-check addresses. Use customer name when delivering.
3. **Efficiency:** load up orders without being late. Plan most efficient route. Use map in app.
4. **Food handling:** do not mix hot and cold bags. Secure drinks (cup holders or floor braced). Pizza trick: bag on its side for extra-large.
5. **Connectivity:** phone charged, notifications enabled, logged in, Wi-Fi OFF at restaurants (free Wi-Fi breaks the app).
6. **Delivery bags:** must use green Valley Eats bag. Second bag requires Courier Relations clearance.
7. **Evaluating courier behavior:** check status accuracy, arrival timing, bag usage, order confirmation, address verification, message responsiveness, food handling.
8. **Addressing issues:** one-off/minor = address directly and respectfully. Repeated patterns = ticket for relations@valleyeats.ca. Not comfortable = ask supervisor or make ticket.

**Escalate If:**
- Repeated pattern of poor behavior (relations@valleyeats.ca)
- Safety-related behavior concerns

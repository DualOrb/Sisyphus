# Email Handling Procedures

**Trigger:** Email received or needs to be sent via Gmail.

**Check First:**
- Email subject line for special types requiring immediate action
- Labels on existing email threads

**Steps:**
1. **Composing:** fill recipient, subject (match ticket subject), body (one topic per line, whitespace between). First name in footer. Hit Send.
2. **Forwarding to restaurants:** NEVER forward original customer email. Create new email, copy only needed details/photos. Erase all customer PII. Label sent email "To be addressed" and "Waiting for Response."
3. **Labels:** "Handled by: [your name]" on every email you touch. "Waiting for Response" when awaiting reply. "Needs to be Addressed" when nothing done yet.
4. **Priority handling:** Immediate = Broken Order, Roger Panic, Fraudulent Customer, Security Alert. High = [Contact] emails, Charge Not Captured, Problem Address. Standard = Notify Error, Suspicious Customer. Routine = general inquiries.
5. **Special emails:** [Contact] = treat as ticket. Charge Not Captured = tell supervisor the order number. Broken Order = tell supervisor. Roger Panic = alert market dispatcher for manual assignment. Fraudulent Customer = alert supervisor immediately. Security Alert = supervisor confirms legitimacy.

**Escalate If:**
- Fraudulent Customer or Security Alert (supervisor immediately)
- Broken Order or Charge Not Captured (supervisor)
- Any email you cannot resolve

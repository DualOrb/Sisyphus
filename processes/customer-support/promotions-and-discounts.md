# Promotions, Discounts, and Gift Cards

**Trigger:** Customer asks about promo codes, discounts, or gift cards.

**Check First:**
- Promotions tab in dispatch for current promo codes
- Discounts tab for active auto-applied discounts
- Gift Cards page if recipient claims they did not receive

**Steps:**
1. **Promo codes:** search Promotions tab. Check: exact match, order meets criteria, valid date range, correct promo type. Do NOT give out promo codes to customers who do not already have one.
2. **Discounts:** auto-applied at checkout (no code needed). Do NOT modify Promotions or Discounts pages.
3. **Gift cards:** customer purchases, recipient gets email with code. If not received: search by recipient email on Gift Cards page, resend via green envelope. Code is case sensitive, may be in spam. Gift card refunds = speak to supervisor. Do NOT modify Gift Cards page.
4. **Dispatch symbols for reference:** green pin = on time. Red pin = late. Red box = 10+ min past pickup. Star = pre-order. Eightball = future-date pre-order. Recycling = re-dispatch. Orange vest = trainee courier. Beer emoji = Smart Serve certified (alcohol delivery).

**Escalate If:**
- Something appears wrong with a promo code itself (flag to supervisor)
- Gift card refund requested (supervisor)
- Customer disputes a discount amount

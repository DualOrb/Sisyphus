# Promotions, Discounts, and Gift Cards

> Source: Dispatch Analyst Guide -- Promotions/Promo Codes, Discounts, Gift Cards sections

## Promotions / Promo Codes

### Accessing Promotions

- Navigate to the **"Promotions"** tab in the right-hand menu of the dispatch system.
- This page lists all current promotional codes.

### What Promo Codes Are

- Promo codes are given to customers from external sources (marketing campaigns, partnerships, etc.).
- They are **not for internal use** by dispatch staff.
- Each promo code entry shows: the code itself, criteria for using it, type of promo code, amount, and start/end dates.

### Troubleshooting a Promo Code Not Working

When a customer reports a promo code is not working:

1. **Do not give out promo codes** to customers who do not already have a good idea of what the code should be.
2. Use the Promotions tab to search for the code.
3. Check the promo code details:
   - Is the code entered correctly (exact match)?
   - Does the customer's order meet the criteria/restrictions?
   - Is the promo code within its valid date range (start and end dates)?
   - Does the promo type match what the customer expects?
4. With this information, identify why the code is not working and explain to the customer.
5. If something appears wrong with the promo code itself, **flag it to a supervisor**.

## Discounts

### Accessing Discounts

- Navigate to the **"Discounts"** tab in the right-hand menu of the dispatch system.

### How Discounts Work

- Discounts are set up by Valley Eats in partnership with restaurants.
- They are **automatically applied** to certain items, certain restaurants, or delivery fees. The customer does not need to enter a code.
- The discount automatically decreases the total cost at checkout.
- Each discount entry shows: the code, criteria, type of discount, amount, and start/end dates.

### Key Rule

- **Do not modify any information on the Promotions or Discounts pages.** Only select personnel on the Valley Eats team may modify these.

## Gift Cards

### How Gift Cards Work

- Customers can purchase gift cards to send to other people.
- The purchaser does **not** need a Valley Eats account.
- Gift cards can be purchased on the Valley Eats website or through the app.
- Process: The purchaser enters the amount, the recipient's email address, and an optional message, then pays.

### Gift Card Details in the System

- Each gift card entry shows all relevant information about the gift code.
- Clicking the **green envelope icon** resends the gift code email to the recipient.
- A **checkmark** appears if the gift card has already been redeemed.

### Troubleshooting Gift Card Issues

When someone calls in saying the recipient did not receive their gift card:

1. Ask for the **recipient's email address**.
2. Use **Ctrl+F** to search for the email on the gift cards page.
3. **Resend** the gift code by clicking the green envelope.
4. Let the caller know:
   - The gift code is **case sensitive**.
   - The email may be in the recipient's **spam folder**.

### Gift Card Refunds

- Valley Eats does **not usually offer refunds on gift cards**.
- If a customer requests a gift card refund, **speak to a supervisor**. Do not process independently.

### Key Rule

- **Do not modify any information on the Gift Cards page.** Only select Valley Eats personnel may make changes.

## Alerts and Symbols Reference

These symbols appear on the dispatch screen and are relevant when handling customer issues:

| Symbol | Meaning |
|---|---|
| Green pin | Order on track, on time, no conflicts |
| Red pin | Order is late and needs attention |
| Red box around order | Order not picked up, 10+ minutes past pickup time -- serious issue |
| Green time boxes | Pickup/delivery predicted to be on time |
| Yellow time boxes | Pickup/delivery will be slightly late |
| Red time boxes | Pickup/delivery will be late |
| Red market background | An order within 11 minutes has not been confirmed by restaurant or courier |
| Star beside order | Pre-order -- customer requested a specific time; take extra care |
| Eightball beside order | Future-date pre-order (e.g., holiday reservation) |
| Recycling symbol | Order was re-dispatched |
| Orange safety vest on courier | Courier in training (fewer than 10 orders or repeated issues); monitor closely |
| Beer emoji on courier | Courier has Smart Serve certification; can deliver alcohol orders (Ontario law requirement) |

### Ping Messages (Pre-Written)

Standard ping messages that can be sent to customers:

- "We have responded to your ticket via email."
- "We apologize for the delay in response. We have responded to your ticket via email."
- "Your order is back on track. Sorry for the inconvenience."
- "Please call Valley Eats at 1 855-610-3287 regarding your order."

### Red Market Background Resolution

When a market has a red background (unconfirmed order within 11 minutes):

- **Call whichever party** (restaurant or courier) has not confirmed the order and have them confirm.
- Be mindful of timing -- if the order was just placed or just assigned, calling immediately may annoy the restaurant or courier.

### Red Box Around Order Resolution

When an order has a red box (10+ minutes past pickup time, not marked "In Bag"):

- This is a **serious issue**. Ensure a courier arrives soon.
- **Communicate delays with the restaurant**, or Valley Eats will almost certainly have to cover the cost of a remake.
- **Reroute the order** to a different courier if needed to minimize time.

### Symbol Legend

New symbols may be added over time. Refer to the **help icon (question mark)** on the top left of the dispatch screen for a current legend of all symbols.
